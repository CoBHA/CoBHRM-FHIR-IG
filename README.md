# bha-test-ig

deviations from [CLAUDE.md](CLAUDE.md):

* uses `package.json` instead of global
* run sushi with `npm run sushi`

try it out:

1. clone the repo, enter the directory

```
$ git clone git@github.com:RatioPBC/bha-test-ig.git

$ cd bha-test-ig
```

2. install sushi

```
$ npm ci
```

3. run sushi

```
$ npm run sushi
```
